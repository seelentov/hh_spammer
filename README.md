# hh_spammer

Инструмент для автоматической рассылки откликов на вакансии [hh.ru](https://hh.ru) по заданным фильтрам поиска. Написан на Go, использует браузерную автоматизацию через [go-rod](https://github.com/go-rod/rod) (Chrome DevTools Protocol).

## Как это работает

1. При запуске открывается браузер с страницей входа hh.ru
2. Вы вручную входите в аккаунт
3. После входа браузер сворачивается и начинается рассылка
4. Программа проходит по всем активным фильтрам, собирает вакансии постранично и откликается на каждую
5. Все отклики сохраняются в SQLite — повторный отклик на ту же вакансию исключён
6. Если вакансия требует ответов на вопросы — браузер разворачивается и ждёт пока вы ответите

## Требования

- Go 1.21+
- Google Chrome или Chromium

## Установка

```bash
git clone <repo>
cd hh_spammer

# Создать файл конфигурации
cp .env.example .env

# Собрать бинарник
go build -o hh_spammer ./cmd/
```

## Конфигурация

Все настройки хранятся в файле `.env` в корне проекта.

| Переменная | Описание | По умолчанию |
|---|---|---|
| `COVER_LETTER` | Текст сопроводительного письма | Стандартный текст |
| `RESUME_TITLE` | Ключевое слово из названия резюме (если их несколько) | Первое резюме |
| `DELAY_MIN` | Минимальная задержка между откликами, секунды | `4` |
| `DELAY_MAX` | Максимальная задержка между откликами, секунды | `9` |

**Пример `.env`:**

```env
COVER_LETTER="Здравствуйте! Меня заинтересовала данная вакансия. Имею опыт разработки на Go 2+ года."

# Если у вас несколько резюме — указать часть названия нужного
RESUME_TITLE=Go разработчик

DELAY_MIN=4
DELAY_MAX=9
```

> Текст письма в `.env` поддерживает `\n` для переносов строк, в том числе внутри кавычек.

## Использование

### Шаг 1 — Добавить фильтры

Откройте hh.ru, настройте поиск вакансий нужными фильтрами и скопируйте URL из адресной строки:

```
https://hh.ru/search/vacancy?text=golang&area=1&experience=between1And3
```

Добавьте его в программу:

```bash
./hh_spammer add-filter "https://hh.ru/search/vacancy?text=golang&area=1&experience=between1And3" "Go middle Москва"
```

Можно добавить несколько фильтров — программа пройдёт по каждому:

```bash
./hh_spammer add-filter "https://hh.ru/search/vacancy?text=golang&area=2" "Go Питер"
./hh_spammer add-filter "https://hh.ru/search/vacancy?text=go+developer&area=1&schedule=remote" "Go remote"
```

### Шаг 2 — Запустить рассылку

```bash
./hh_spammer run
```

Откроется браузер. Войдите в аккаунт hh.ru. После входа браузер свернётся и начнётся рассылка.

## Команды

### `run` — запустить рассылку

```bash
./hh_spammer run [флаги]
```

| Флаг | Описание |
|---|---|
| `-v`, `--visible` | Не сворачивать браузер (удобно для отладки) |
| `-s`, `--skip` | Пропускать вакансии с вопросами для отклика |
| `-t`, `--timeout` | Таймаут ожидания ответов на вопросы (по умолч. `2m`) |

**Примеры:**

```bash
# Обычный запуск
./hh_spammer run

# Браузер всегда виден
./hh_spammer run -v

# Пропускать вакансии с вопросами
./hh_spammer run -s

# Ждать ответов на вопросы не более 90 секунд
./hh_spammer run -t 90s
```

### `add-filter` — добавить фильтр

```bash
./hh_spammer add-filter <url> [название]
```

```bash
./hh_spammer add-filter "https://hh.ru/search/vacancy?text=golang&area=1" "Go Москва"
```

### `list-filters` — список фильтров

```bash
./hh_spammer list-filters
```

```
ID   Статус   Название                       URL
─────────────────────────────────────────────────────────────
1    активен  Go Москва                      https://hh.ru/search/vacancy?text=golang&area=1
2    откл.    Go Питер                       https://hh.ru/search/vacancy?text=golang&area=2
```

### `disable-filter` / `enable-filter` — отключить / включить фильтр

```bash
./hh_spammer disable-filter 2
./hh_spammer enable-filter 2
```

### `delete-filter` — удалить фильтр

```bash
./hh_spammer delete-filter 2
```

### `list-applied` — история откликов

```bash
# Последние 50 (по умолчанию)
./hh_spammer list-applied

# Последние 200
./hh_spammer list-applied 200
```

```
[✓] 2026-04-19 12:34  Рога и Копыта — Go разработчик     https://hh.ru/vacancy/123456
[✓] 2026-04-19 12:28  Acme Corp — Backend Engineer        https://hh.ru/vacancy/654321
[✗] 2026-04-19 12:21  Foo Ltd — Golang Dev                https://hh.ru/vacancy/999999  [кнопка не найдена]
```

## Поведение при разных сценариях

| Ситуация | Поведение |
|---|---|
| Стандартная вакансия (попап с письмом) | Заполняет письмо, нажимает «Отправить» |
| Отклик отправлен сразу (скролл к форме письма) | Заполняет письмо в форме ниже на странице |
| Уже откликался на эту вакансию | Пропускает (проверяет и по БД, и по статусу на сайте) |
| Внешняя вакансия (ведёт на сторонний сайт) | Пропускает |
| Вакансия с вопросами | Разворачивает браузер, ждёт пока вы ответите (или пропускает при `-s`) |

## База данных

SQLite-файл `hh_spammer.db` создаётся автоматически в текущей директории. Содержит две таблицы:

- **filters** — список фильтров поиска (URL, название, флаг активности)
- **applied** — история откликов (ID вакансии, URL, компания, должность, дата, статус, текст ошибки)
